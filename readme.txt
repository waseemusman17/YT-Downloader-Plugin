Download YouTube videos directly from the WordPress admin panel and store them in your Media Library.

== Description ==
Admin users can:
- Paste a YouTube video URL.
- Download it using yt-dlp.
- Store it in the Media Library automatically.

== Installation ==
1. Ensure `yt-dlp` is installed on your server and available in the system PATH.
2. Upload the plugin folder to /wp-content/plugins/.
3. Activate the plugin via WordPress admin.
4. Navigate to Tools > YouTube Downloader to use.
